En esta página se expone el estudio de patrones UX (experiencia de usuario), sus categorías, sub categorías y de qué manera estos imparten un modo de interacción entre el usuario y un servicio.
La UX se presenta como una forma de jerarquizar y catalogar el comportamiento del servicio. De esta correlación se desprenden las buenas y malas experiencias.
